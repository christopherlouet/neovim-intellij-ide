# Release v1.0.0 — Neovim IntelliJ-like IDE

Première release **stable** : une configuration Neovim orientée “IDE” inspirée d’IntelliJ, prête à être publiée
et consommée via GitHub.

## Highlights
- Expérience IDE : navigation projet, diagnostics centralisés, complétion/snippets, UI moderne.
- LSP via Mason + configuration native Neovim.
- Formatting & qualité de code via none-ls (avec garde-fous).

## Fixes
- Correction du crash none-ls lorsque `eslint_d` n’est pas disponible en builtin :
  chargement via `none-ls-extras` + fallback vers `eslint`.

## Repository hardening
- CI de lint (shellcheck, stylua, luacheck, markdownlint).
- Fichiers OSS standards : LICENSE, CONTRIBUTING, CHANGELOG, templates Issues/PR.
- Install scripts plus robustes (dont `--dry-run` sûr).
